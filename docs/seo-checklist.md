# SEO Optimization Checklist

## Page-Level SEO
- [ ] Title tag: 50-60 characters, includes primary keyword
- [ ] Meta description: 150-160 characters, compelling call-to-action
- [ ] URL structure: Clean, readable, includes keyword
- [ ] H1 tag: One per page, includes primary keyword
- [ ] H2-H6 tags: Logical hierarchy, includes related keywords
- [ ] Image alt text: Descriptive, includes relevant keywords
- [ ] Internal linking: Links to related content
- [ ] External linking: Links to authoritative sources
- [ ] Mobile-friendly: Responsive design
- [ ] Page speed: < 3 seconds load time

## Content Quality
- [ ] Original content (no duplicate/copied text)
- [ ] Minimum 800-1500 words for review pages
- [ ] Proper grammar and spelling
- [ ] Easy-to-read formatting (short paragraphs, bullet points)
- [ ] Multimedia (images, videos if applicable)
- [ ] Accurate information with citations
- [ ] Updated publish/modified dates

## Technical SEO
- [ ] robots.txt properly configured
- [ ] sitemap.xml updated with new pages
- [ ] Canonical URL set correctly
- [ ] Schema markup (if applicable)
- [ ] HTTPS enabled (GitHub Pages default)
- [ ] Open Graph tags for social sharing

## Affiliate Best Practices
- [ ] Clear affiliate disclosure
- [ ] rel="nofollow noopener" on affiliate links
- [ ] Value-driven content (not just product pitches)
- [ ] Honest pros and cons
- [ ] Multiple product options when relevant
